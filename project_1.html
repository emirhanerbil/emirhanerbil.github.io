About Project

This project is my graduation project and it came 4th in the competition.
**The project's server is not currently live.**

- The project is used via the chrome extension that I have created.

- While you are in the Youtube video tab, you open the extension and click the test button.

- With this button, I send a post request to my fastapi server running on gcp using javascript for the youtube link.

- Thanks to this link coming to my server, I pull the subtitles of that video using the python library and make them suitable for inserting into my ai model.

- If I talk about these models, these models have 5 output classes. Profanity, racism, sexism, insult and none.

- I prepared the dataset myself according to these output values. This dataset contains approximately 60 thousand rows.

- I trained 3 models with this dataset. These are svc, lstm and the bert model that I fine-tuned.

- I insert the sentences spoken in the video into these 3 models and have them decide with the voting system.

- The operations as NLP are up to this point. For image classification, I download the video in the youtube link to which the post request is sent to the server.

- Then I send the images to my model at 10 frames per second. The model I used here is YOLOV8m. I downloaded the dataset I used from kaggle.

- It contains about 5 thousand knife and gun photos. After the ai processes are completed, database processes start. I do it to save some statistics like how many videos i processed etc.

- I use mongodb as a database. Finally, a unique endpoint is created. This endpoint is the youtube video unique id that I extracted from the link in the request.

- Then I return this endpoint as the response of the post request that was sent at the beginning and we can access it from the chrome extension.
